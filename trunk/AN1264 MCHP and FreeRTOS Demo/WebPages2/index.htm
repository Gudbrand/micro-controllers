Microchip Metering Demo
 
Overview Monitor Billing Configuration

 

This application demonstrates using the Microchip TCP/IP, QVGA and MiWi Stacks together with the FreeRTOS.org open source real-time operating system.

The metering demo allows for local control of the electricity and gas functions allowing a user to observe the amount of each energy source being consumed and the total cost. The user can turn on and off the energy supplies mimicking the behaviour of a real system.

Remote monitoring and control of the hardware on the Explorer16 board via the use of the Ethernet connection is supported. A remote user can also isolate and turn the supplies on and off as well as change the billing parameters for both energy sources using the Billing tab. Via the use of the AJAX interface realtime data can be viewed in the Monitor tab.
